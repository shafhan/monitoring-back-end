1. File 1.3_gambar-1(mengedit php diterminal).png

    Setelah saya membuat file php baru, saya menulikan perintah di terminal
    
    nano index.php
    
    perintah tersebut digunakan untuk merubah file index.php langsung diterminal, setelah saya menuliskan sintaks php di index.php, saya menuliskan perintah 
    
    php index.php

    perintah php <name file> digunakan untuk menjalankan sintaks php atau menampilkan output dari sintaks php yang sudah di buat tampil di terminal
